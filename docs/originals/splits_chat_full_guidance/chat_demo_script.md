# Splits Network Chat — Demo Script (Full Feature)

Accounts:
- Candidate (applicant.network)
- Recruiter (splits.network)
- Company user (splits.network)

Flow:
1. Recruiter messages candidate from application → candidate sees Request.
2. Candidate accepts → realtime thread opens.
3. Typing indicator + presence dots.
4. Read receipt appears after candidate opens message.
5. Attach a PDF → “Scanning…” → “Available”.
6. Candidate blocks → recruiter send fails neutrally.
7. Recruiter ↔ company conversation updates in realtime with job context banner.
