Key Integration Points / Potential Failure Points

1. GPT Builder Configuration

Instructions must be <8000 characters. OpenAPI schema must be imported correctly. OAuth authentication must be configured with Clerk credentials.

2. OAuth / Clerk Token Exchange

ChatGPT must exchange tokens with Clerk. If the OAuth callback URL or client credentials are wrong, all authenticated actions fail silently.

3. API Gateway Routing

/api/v1/gpt/* routes must be configured in the gateway. POST/DELETE body re-serialization must be handled correctly.

4. RabbitMQ Connection

GPT Service must have an active RabbitMQ connection to publish application.created and application.stage_changed events.

5. AI Service Consumer

AI Service must be running and listening for events where stage = 'ai_review'. If down, applications stall at ai_review indefinitely.
