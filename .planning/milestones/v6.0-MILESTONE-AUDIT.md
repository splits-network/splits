## Integration Gaps — All Fixed

### Fixed: useStandardList Portal Gateway Hard-Code
Added `clientFactory` option to `UseStandardListOptions`. Admin wrapper now passes `createAdminClient` so all 17+ admin list pages route through admin-gateway.

### Fixed: shared-charts Missing from transpilePackages
Added `@splits-network/shared-charts` to `apps/admin/next.config.mjs` transpilePackages.

### Fixed: Content Pages Admin Routes Missing
Created `services/content-service/src/v2/admin/routes.ts` with `GET /admin/pages`, `GET /admin/pages/:id`, and `GET /admin/counts`.

### Fixed: Escrow Release Endpoint Missing
Added `POST /admin/escrow/:id/release` to billing-service admin routes (repository + service + route).

### Fixed: Settings Path Prefix
Updated settings page and form to use `/admin/identity/admin/settings` (proxy-compatible path). No backend persistence yet — accepted as tech debt.
