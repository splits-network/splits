    **gateway calls.ts**: Add a specific `app.post('/api/v2/calls/exchange-token', ...)` route BEFORE the existing `app.all('/api/v2/calls/*', ...)` catch-all. This route must NOT use `buildAuthHeaders(request)` -- pass empty headers `{}` instead. It proxies `request.body` to `callService().post('/api/v2/calls/exchange-token', request.body, correlationId, {})`. Import `getCorrelationId` from `./common` (already imported). The route does not need `noAuth()` preHandler because auth is handled at the global level in index.ts.
