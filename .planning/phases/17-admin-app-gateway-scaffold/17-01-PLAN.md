<objective>
Create the admin-gateway Fastify service: a thin authenticated HTTP proxy that forwards requests to existing domain services. It verifies admin Clerk JWTs, checks isPlatformAdmin via resolveAccessContext, and uses @fastify/http-proxy to forward requests with context headers.

Purpose: Provides the backend gateway for the admin app, keeping admin auth isolated from portal's api-gateway.
Output: A working Fastify service in services/admin-gateway/ that starts, authenticates, and proxies.
</objective>

    - Name: `@splits-network/admin-gateway`
    - Scripts: `dev` (tsx watch src/index.ts), `build` (tsc), `start` (node dist/index.js)
    - Dependencies: fastify, @fastify/http-proxy, @fastify/rate-limit, @clerk/backend, ioredis, @supabase/supabase-js, and all shared packages (@splits-network/shared-config, shared-logging, shared-fastify, shared-access-context, shared-types) as workspace:*
    - devDependencies: typescript, tsx, @types/node
    - Match version patterns from api-gateway's package.json

    - Extend `../../tsconfig.base.json` (same as api-gateway)
    - outDir: ./dist, rootDir: ./src
    - Include src/**/*.ts

    **src/auth.ts:**
    - Export `AdminAuthMiddleware` class (see Pattern 3 in RESEARCH.md)
    - Constructor takes: clerkSecretKey, supabaseUrl, supabaseServiceRoleKey
    - Creates Clerk client via `createClerkClient({ secretKey })`
    - Creates Supabase client with service role key (NOT anon key -- see Pitfall 5)
    - `createMiddleware()` returns a Fastify preHandler that:
      1. Extracts Bearer token from Authorization header (throw UnauthorizedError if missing)
      2. Verifies token via `verifyToken(token, { secretKey: this.clerkSecretKey })` -- store secretKey as instance property
      3. Calls `resolveAccessContext(this.supabase, verified.sub)` to get access context
      4. Checks `context.isPlatformAdmin` -- throw ForbiddenError if false
      5. Attaches `request.adminAuth = { clerkUserId: verified.sub, userId: context.identityUserId, isPlatformAdmin: true }` to request
    - Import UnauthorizedError and ForbiddenError from @splits-network/shared-fastify
    - Import resolveAccessContext from @splits-network/shared-access-context
  </action>

  </verify>
  <done>
