    - Import ClerkProvider from @clerk/nextjs
    - Read NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY from process.env, throw if missing
    - Wrap html > body with ClerkProvider and QueryProvider
    - Body classes: `flex flex-col min-h-screen bg-base-300`
    - Include FontAwesome Script tag (same as portal -- see Pattern 5 in RESEARCH.md)
    - Import globals.css
    - Metadata: title "Splits Network Admin", description "Admin Portal"

    **src/app/page.tsx** (public landing):
    - Server component
    - Branded landing page: Splits Network logo (or just text "Splits Network"), "Admin Portal" title, "Sign In" button/link to /sign-in
    - Clean, centered layout. Use DaisyUI card component.
    - No auth check needed -- this is the public entry point

    **src/app/unauthorized/page.tsx:**
    - Server component
    - Message: "You don't have admin access to this application."
    - Link to /sign-in with text "Sign in with a different account"
    - Simple centered layout with DaisyUI card

    **src/app/(auth)/layout.tsx:**
    - Simple centered layout for auth pages (flex items-center justify-center min-h-screen)

    **src/app/(auth)/sign-in/[[...sign-in]]/page.tsx:**
    - Import SignIn from @clerk/nextjs
    - Render `<SignIn />` component (Clerk handles the UI)
    - Mirror portal's sign-in page pattern

    **src/app/secure/layout.tsx** (THE CRITICAL GATE):
    - Server component (no 'use client')
    - Import `auth` from `@clerk/nextjs/server`
    - Import `redirect` from `next/navigation`
    - Get userId and getToken from `await auth()`
    - If no userId, redirect to /sign-in
    - Get token via `await getToken()`
    - Fetch user profile from admin-gateway: `${process.env.NEXT_PUBLIC_ADMIN_GATEWAY_URL}/admin/identity/users/me` with Bearer token
    - Check response.data.is_platform_admin === true
    - If not admin, redirect to /unauthorized
    - Render children if admin (see Pattern 2 in RESEARCH.md)
    - Wrap in try/catch -- any fetch error redirects to /unauthorized

    **src/app/secure/page.tsx** (welcome placeholder):
    - Server component
    - Import `currentUser` from `@clerk/nextjs/server`
    - Display: "Welcome, {user.firstName}" heading
    - Brief text: "Admin dashboard coming soon."
    - Include UserButton from @clerk/nextjs for logout functionality (this is a client import, so either make this a client component or extract the UserButton into a small client component)
    - Clean layout with DaisyUI card
