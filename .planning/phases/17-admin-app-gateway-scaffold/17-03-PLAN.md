       Build admin: `pnpm --filter @splits-network/admin build`
       Prune: `pnpm install --prod --frozen-lockfile`
    4. `production` stage: node:22-alpine, corepack enable pnpm.
       Copy workspace config (package.json, pnpm-workspace.yaml).
       Copy node_modules from build.
       Copy built shared packages (package.json + dist/ for each).
       Copy apps/admin/package.json, .next/, public/ (if exists), next.config.mjs.
       EXPOSE 3200. CMD `pnpm --filter @splits-network/admin start`

       - packages/shared-config/package.json
       - packages/shared-fastify/package.json
       - packages/shared-logging/package.json

       - packages/shared-access-context/package.json
       - services/admin-gateway/package.json

       Run `pnpm install --frozen-lockfile` again.
       Build shared packages: shared-config, shared-types, shared-logging, shared-fastify, shared-access-context.
       Build service: `pnpm --filter @splits-network/admin-gateway build`
       Prune: `pnpm install --prod --frozen-lockfile`
