  <action>
    Create `packages/shared-hooks/` directory and package scaffolding.

    **package.json:** Follow shared-ui pattern.
    - Name: `@splits-network/shared-hooks`
    - Build: `tsc -b` (same as shared-ui, NOT tsup)
    - Main: `./dist/index.js`, Types: `./dist/index.d.ts`
    - Dependencies: `@splits-network/shared-types: workspace:*`, `@splits-network/shared-api-client: workspace:*`, `@splits-network/shared-ui: workspace:*`
    - peerDependencies: `react: ^19.0.0`, `react-dom: ^19.0.0`, `next: ^16.0.0`, `@tanstack/react-query: *`
    - devDependencies: `@types/react`, `@types/react-dom`, `typescript`

    **tsconfig.json:** Follow shared-ui pattern exactly (extends ../../tsconfig.base.json, jsx: react-jsx, outDir: ./dist, rootDir: ./src, lib: ES2022 + DOM).

    **api-client.ts:** Extract from `apps/portal/src/lib/api-client.ts` but restructured per decisions:
    - Remove the portal-specific `ApiClient` class with its business methods (`getCurrentRecruiter`, `getCurrentSubscription`). Those stay in portal.
    - Create a generic `AppApiClient` class that wraps `SplitsApiClient` with the same get/post/patch/put/delete interface (passing through to underlying client).
    - Export two factory functions:
      - `createPortalClient(token: string)` - creates an AppApiClient configured for portal (base URL from `NEXT_PUBLIC_API_URL` or `NEXT_PUBLIC_API_GATEWAY_URL` env vars, same as current SplitsApiClient auto-detection)
      - `createAdminClient(token: string)` - creates an AppApiClient configured for admin (base URL from `NEXT_PUBLIC_ADMIN_GATEWAY_URL` env var, falls back to `http://localhost:3001`)
    - Token is passed as argument (no Clerk import). No Clerk coupling in this package.
    - Also export `createUnauthenticatedClient()` for public routes.
    - Re-export `ApiError` and `ApiResponse` from `@splits-network/shared-api-client` for convenience.

    **toast-context.tsx:** Copy from `apps/portal/src/lib/toast-context.tsx` verbatim. The toast context is framework-agnostic (pure React + DaisyUI classes). No changes needed. Export `ToastProvider`, `useToast`, `Toast`, `ToastType`.

    **use-standard-list.ts:** Copy from `apps/portal/src/hooks/use-standard-list.ts` with these import changes:
    - Change `import { ApiClient, createAuthenticatedClient } from '@/lib/api-client'` to import `AppApiClient` and `createPortalClient` from `./api-client`. NOTE: The hook uses both `new ApiClient()` (unauthenticated) and `createAuthenticatedClient(token)` - map these to `createUnauthenticatedClient()` and `createPortalClient(token)` respectively from `./api-client`.
    - Change `import { useAuth } from '@clerk/nextjs'` to accept `getToken` as a parameter in the hook options instead. Add `getToken?: () => Promise<string | null>` to `UseStandardListOptions`. When `requireAuth` is true and no `getToken` is provided, throw an error. This removes Clerk coupling from the shared package.
    - Change the 6 re-exports of `@/components/standard-lists/*` to re-export from `@splits-network/shared-ui` instead (after Task 2 moves them there). The re-exports are: SearchInput, PaginationControls, EmptyState, LoadingState, ErrorState, ViewModeToggle.
    - Change `import type { StandardListParams, StandardListResponse, PaginationResponse } from '@splits-network/shared-types'` stays the same (already a shared package import).
    - Remove the `useRouter`, `usePathname`, `useSearchParams` imports from `next/navigation`. Instead, accept optional `router` integration via options: add `urlSync?: { searchParams: URLSearchParams; pathname: string; replace: (url: string, options?: any) => void }` to options. When `syncToUrl` is true and `urlSync` is provided, use it; when `syncToUrl` is true and `urlSync` is NOT provided, silently disable URL sync (no crash). This removes the hard Next.js coupling.
    - IMPORTANT: The `useQuery` and `useQueryClient` imports from `@tanstack/react-query` stay as-is (peer dependency).

    **index.ts:** Barrel exports:
    ```typescript
    // API Client
    export { AppApiClient, createPortalClient, createAdminClient, createUnauthenticatedClient } from './api-client';
    export type { ApiError, ApiResponse } from '@splits-network/shared-api-client';

    // Toast
    export { ToastProvider, useToast } from './toast-context';
    export type { Toast, ToastType } from './toast-context';

    // Standard List Hook
    export { useStandardList } from './use-standard-list';
    export type { UseStandardListOptions, UseStandardListReturn, FetchParams, FetchResponse } from './use-standard-list';

    // Re-export standard list UI components from shared-ui for convenience
    export { SearchInput, PaginationControls, EmptyState, ErrorState, ViewModeToggle } from '@splits-network/shared-ui';
    ```
