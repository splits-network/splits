  <files>services/ats-service/src/v2/companies/repository.ts</files>
  <action>
After fetching companies in `findCompanies`, add a secondary query to compute `open_roles_count` and `avg_salary` for the returned set.

**In `findCompanies`** — after `const { data, error, count } = await query;` and before the return statement:

1. Extract company IDs: `const companyIds = (data || []).map((c: any) => c.id);`
2. If `companyIds.length > 0`, query jobs table:
   ```typescript
   const { data: jobRows } = await this.supabase
       .from('jobs')
       .select('company_id, status, salary_min, salary_max')
       .in('company_id', companyIds);
   ```
3. Build a stats map by iterating `jobRows`:
   - For each job where `status === 'active'`, increment `open_roles_count`
   - For each active job where `salary_min != null && salary_max != null`, accumulate `(salary_min + salary_max) / 2` into a sum and increment a salary count
4. Map over `data` to merge stats:
   ```typescript
   const enrichedData = (data || []).map((c: any) => ({
       ...c,
       open_roles_count: statsMap[c.id]?.open_roles_count ?? 0,
       avg_salary: statsMap[c.id]?.salary_count > 0
           ? Math.round(statsMap[c.id].salary_sum / statsMap[c.id].salary_count)
           : null,
   }));
   ```
   Use `Math.round` to return a clean integer for avg_salary (no floating point decimals in currency).
5. Return `{ data: enrichedData, total: count || 0 }` instead of the raw data.

If `companyIds.length === 0`, return early with `{ data: [], total: 0 }` (no secondary query needed).

**In `findCompany`** — after fetching the single company and before `return data;`:

1. Query jobs for this single company:
   ```typescript
   const { data: jobRows } = await this.supabase
       .from('jobs')
       .select('status, salary_min, salary_max')
       .eq('company_id', id);
   ```
2. Filter active jobs and compute stats in-memory:
   ```typescript
   const activeJobs = (jobRows || []).filter((j: any) => j.status === 'active');
   const salaryJobs = activeJobs.filter((j: any) => j.salary_min != null && j.salary_max != null);
   data.open_roles_count = activeJobs.length;
   data.avg_salary = salaryJobs.length > 0
       ? Math.round(salaryJobs.reduce((sum: number, j: any) => sum + (j.salary_min + j.salary_max) / 2, 0) / salaryJobs.length)
       : null;
   ```
3. Return `data` as before.

**Important:** Only count jobs where `status === 'active'` for `open_roles_count`. Only include active jobs with non-null salary_min AND salary_max for `avg_salary`. When no qualifying jobs exist, `open_roles_count: 0` and `avg_salary: null`.
  </action>
