  <name>Task 1: Create scheduling schema migration</name>
  <files>supabase/migrations/20260308000001_extend_interviews_for_scheduling.sql</files>
  <action>
Create a migration that extends the interviews schema for scheduling:

1. **Add columns to `interviews` table:**
   - `calendar_event_id TEXT` — external calendar event ID (Google/Microsoft)
   - `calendar_connection_id UUID REFERENCES oauth_connections(id)` — which calendar connection was used
   - `meeting_platform TEXT NOT NULL DEFAULT 'splits_video'` — enum-like: 'splits_video', 'google_meet', 'microsoft_teams'
   - `meeting_link TEXT` — external meeting URL (for Google Meet / Teams)
   - `original_scheduled_at TIMESTAMPTZ` — preserved when rescheduled
   - `reschedule_count INT NOT NULL DEFAULT 0`
   - `reschedule_requested_by TEXT` — 'interviewer' or 'candidate'
   - `reschedule_requested_at TIMESTAMPTZ`
   - `reschedule_notes TEXT` — reason for reschedule request

2. **Create `user_calendar_preferences` table:**
   ```sql
   CREATE TABLE user_calendar_preferences (
       id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
       user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE UNIQUE,
       connection_id UUID REFERENCES oauth_connections(id),
       primary_calendar_id TEXT,
       additional_calendar_ids TEXT[] DEFAULT '{}',
       working_hours_start TIME NOT NULL DEFAULT '09:00',
       working_hours_end TIME NOT NULL DEFAULT '17:00',
       working_days INT[] NOT NULL DEFAULT '{1,2,3,4,5}',
       timezone TEXT NOT NULL DEFAULT 'America/New_York',
       created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
       updated_at TIMESTAMPTZ NOT NULL DEFAULT now()
   );
   ```
   - `working_days` uses ISO day numbers (1=Monday, 7=Sunday)
   - Add updated_at trigger
   - Add index on user_id

3. **Create `interview_reschedule_requests` table:**
   ```sql
   CREATE TABLE interview_reschedule_requests (
       id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
       interview_id UUID NOT NULL REFERENCES interviews(id) ON DELETE CASCADE,
       requested_by TEXT NOT NULL, -- 'candidate' or 'interviewer'
       requested_by_user_id UUID REFERENCES users(id),
       status TEXT NOT NULL DEFAULT 'pending', -- 'pending', 'accepted', 'declined'
       proposed_times JSONB NOT NULL DEFAULT '[]', -- array of {start, end} ISO strings
       accepted_time TIMESTAMPTZ,
       notes TEXT,
       created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
       updated_at TIMESTAMPTZ NOT NULL DEFAULT now()
   );
   ```
   - Add index on interview_id
   - Add updated_at trigger

4. **Add indexes:**
   - `CREATE INDEX idx_interviews_calendar_event ON interviews(calendar_event_id) WHERE calendar_event_id IS NOT NULL;`
  </action>
