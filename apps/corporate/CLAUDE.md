# Corporate App

Marketing / corporate website — Next.js 16, App Router.

## Key Sections

```
src/app/cms/          # CMS-driven content pages
src/app/contact/      # Contact page
src/app/showcase/     # Design showcase
src/app/privacy-policy/ & cookie-policy/
```

## Skills

- `/ui` — DaisyUI component patterns, theme tokens
- `/seo` — SEO audit & optimization (top priority for this app)
- `/aio` — AI optimization for marketing content
- `/basel` — Basel design system migration

## Notes

- Primarily static/SSR — SEO and performance are top priorities
- Has `robots.ts` and `sitemap.ts` for search engines
- Minimal auth — mostly public content
