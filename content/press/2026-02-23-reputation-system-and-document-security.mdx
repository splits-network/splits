Trust and security are paramount in our marketplace. Today we're introducing a reputation management system and bolstering our document security measures.

## New Features

A new **Company Reputation Management System** is now live. This system helps build trust across the network by tracking and displaying reputation metrics for companies.

We've significantly enhanced security with **Document Virus Scanning**. All uploaded files are now automatically scanned using ClamAV, and we've added a **Scan Status** indicator so you can be confident that files are safe to open.

To help you track performance, we've added a **Marketplace Metrics Aggregation Job**, which will power new analytics dashboards with richer data.

## Improvements

We've introduced **HMAC-signed confirmation tokens** for improved security in our verification flows. Additionally, we've updated file upload restrictions to strictly allow only PDF and Word documents, ensuring consistency and safety for all attachments.
